## 1. 顶部导航栏 (Dark Frosted Glass Navbar)

> *   **视觉规范**：导航栏改为深色调，使用深靛蓝或深炭灰 (`background: rgba(15, 23, 42, 0.8)`)。
> *   **质感要求**：必须具备**磨砂玻璃效果**：`backdrop-filter: blur(16px)`，并配合 `border-bottom: 1px solid rgba(255, 255, 255, 0.1)`。
> *   **文字与图标**：所有导航链接、搜索框图标和用户信息文字统一改为白色 (`#FFFFFF`)，搜索框背景改为半透明黑色 (`rgba(0, 0, 0, 0.2)`)。
> 

## 2. 动态弥散背景 (Mesh Gradient Background)
> *   **背景实现**：移除纯白背景。使用 CSS `radial-gradient` 实现多色彩弥散渐变。
> *   **色彩参数**：混合浅蓝 (`#E0F2FE`)、浅粉 (`#FDF2F8`) 和浅黄 (`#FEFCE8`)。
> *   **动效**：添加一个 15s 的无限循环动画，让色块位置产生轻微漂移，营造“呼吸感”。

## 3. 布局与渐变阴影分割线 (Divider Logic)
> *   **中缝设计**：在“失物招领栏”和“失物寻物栏”之间添加一条**渐变阴影垂直线**。
> *   **CSS 实现建议**：
>     ```css
>     .divider {
>       width: 1px;
>       background: linear-gradient(to bottom, transparent, rgba(0,0,0,0.08), transparent);
>       box-shadow: 0 0 15px rgba(0,0,0,0.05);
>     }
>     ```
> *   **间距对齐**：确保左右两栏有足够的 `padding` (约 40px)，让分割线在视觉上起到轻盈的区隔作用。
>

## 4. 校园多巴胺卡片 (Vibrant Bento Cards)
> *   **形状**：卡片圆角提升至 `24px`，显得更圆润友好。
> *   **投影**：使用多重软阴影 `box-shadow: 0 10px 15px -3px rgba(0,0,0,0.05), 0 4px 6px -2px rgba(0,0,0,0.02)`。
> *   **交互**：鼠标悬停时，卡片需平滑上浮 `8px` 且带有轻微缩放 `scale(1.02)`，过渡时间 `0.4s`。
> *   **标签颜色**：使用高饱和度色彩区分状态（如：寻物中-玫瑰红，已找到-活力橙）。
>
