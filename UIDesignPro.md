## 🚀 DeepSeek 专用：失物招领网站“活泼校园版”重构指令

> **DeepSeek，请按照以下“活泼有趣校园风”规范，重写整个项目的 CSS 样式，并加入暗黑模式支持：**

### 1. 弥散呼吸背景 (Mesh Gradient & Animation)
*   **日间模式：** 使用 `#E0F2FE`(天蓝), `#FFEDD5`(淡橙), `#F0FDF4`(薄荷绿) 三色弥散。
*   **暗黑模式：** 使用 `#1E1B4B`(深靛), `#312E81`(暗紫), `#4338CA`(霓虹蓝)。
*   **要求：** 背景不使用静态图，请用 `radial-gradient` 编写，并添加一个 20s 的缓慢位移动效，模拟云雾呼吸。

### 2. 玻璃拟态卡片 (Bubble-Glass Style)
*   **圆角：** 极度圆润。卡片 `border-radius` 统一设为 `24px`。
*   **质感：** 背景 `rgba(255, 255, 255, 0.4)`，暗黑模式下为 `rgba(30, 41, 59, 0.5)`。
*   **边框：** 2px 的半透明白边，模拟肥皂泡的边缘感。
*   **微交互：** 鼠标悬停时，卡片不仅仅是上浮，还要配合一个微小的旋转 `rotate(1deg)`，增加“活泼感”。

### 3. “多巴胺”色彩系统 (Campus Vibrant Colors)
*   **主色调：** 使用高饱和度的 `Candy Blue (#3B82F6)`。
*   **状态标签：** 
    *   “寻物中”：使用波浪纹背景的粉红色 (`#FB7185`)。
    *   “已找到”：使用充满活力的活力橙 (`#FB923C`)。
*   **字体：** 优先使用圆润的无衬线体（如 `Quicksand` 或 `PingFang SC SemiBold`），并增加 `letter-spacing: 0.05em`。

### 4. Bento Grid 布局优化
*   **非对称布局：** 页面容器采用 `display: grid; gap: 24px;`。
*   **主次分明：** 将主列表（如 **屏幕截图 2026-05-05 145034.jpg** 中的内容区）设置为 `grid-column: span 2`，右侧加入一个小型的“校园热搜/丢拾趋势”看板。
*   **卡片内容：** 图片加上 `hover: grayscale(0) scale(1.1)` 的过渡，默认带一点轻微的模糊或灰度。

### 5. 暗黑模式 (Dark Mode) 逻辑实现
*   请生成两套核心变量：`:root`（日间）和 `[data-theme='dark']`（暗色）。
*   暗色模式下，文字使用低对比度的灰白 `#E2E8F0`，防止刺眼。
*   **切换开关：** 请帮我编写一个可点击的切换按钮（Sun/Moon 图标），并添加平滑的 `transition: all 0.5s cubic-bezier(0.4, 0, 0.2, 1)`。
